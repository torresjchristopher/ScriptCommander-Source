Data Sovereignty by Design.

Shortcut CLI is engineered to be invisible to your data. We do not want your information.

Local-Only Execution

All scripts, file histories, and recent document caches are read directly from your filesystem and exist only in volatile memory during runtime. No local data is ever mirrored to a remote database.

Zero Telemetry Architecture

Shortcut CLI contains no analytics engines, no tracking pixels, and no "call home" functions. We do not track usage patterns, execution counts, or user identities.

Identity-Free Workflow

There are no user accounts, no email signups, and no authentication requirements. Your identity is never linked to the software, ensuring total anonymity in your automation tasks.

Open-Source Verifiability

Every line of Shortcut CLI's source code is publicly auditable. We encourage security researchers to verify our zero-data collection claims directly on GitHub.
